WinForms – Gomb események

Click · MouseHover · Width növelés — hiányzik a portfólióból
01
Form1.cs – három gomb, három esemény
Gomb1: kattintásra pirossá válik. Gomb2: kattintásra 10px-lel szélesebb lesz. Gomb3: ráhúzásra sárgává válik.
public partial class Form1 : Form
{
    public Form1()
    {
        InitializeComponent();
    }

    // Gomb1 – kattintásra vörös lesz
    private void button1_Click(object sender, EventArgs e)
    {
        button1.BackColor = Color.Red;
    }

    // Gomb2 – kattintásra 10 px-szel szélesebb lesz
    private void button2_Click(object sender, EventArgs e)
    {
        button2.Width += 10;
    }

    // Gomb3 – ráhúzásra sárga lesz
    private void button3_MouseHover(object sender, EventArgs e)
    {
        button3.BackColor = Color.Yellow;
    }
}
⚠️ WinForms projekt. Három Button szükséges a Designerben. A button3-hoz a MouseHover eseményt kell bekötni (nem MouseMove, nem Click!).
