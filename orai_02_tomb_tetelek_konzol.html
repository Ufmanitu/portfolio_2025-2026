Tömb-tételek – konzol változatok

Keresés · Megszámlálás · Összegzés · Eldöntés · Kiválasztás — hiányzik a portfólióból
ℹ️ A portfólió #12-es feladata ezeket WinForms GUI-val mutatja be. Ez a fájl az alap, metódusos konzol változatokat tartalmazza, ahogy az órákon megírtuk.
01
Keresés (lineáris)
Megkeresi a keresett értéket a tömbben; visszaadja az indexét, vagy -1-et ha nincs meg.
static int Kereses(int[] T, int keresett)
{
    for (int i = 0; i < T.Length; i++)
        if (T[i] == keresett)
            return i;
    return -1;
}
02
Megszámlálás
Megszámolja a páros elemek számát a tömbben.
static int Megszamlalas(int[] T)
{
    int db = 0;
    for (int i = 0; i < T.Length; i++)
        if (T[i] % 2 == 0)
            db++;
    return db;
}
03
Összegzés
Összeadja a tömb összes elemét és visszaadja az összeget.
static int Osszegzes(int[] T)
{
    int s = 0;
    for (int i = 0; i < T.Length; i++)
        s += T[i];
    return s;
}
04
Eldöntés
Eldönti, hogy van-e negatív szám a tömbben. Igaz, ha igen; hamis, ha nincs.
static bool Eldontes(int[] T)
{
    for (int i = 0; i < T.Length; i++)
        if (T[i] < 0)
            return true;
    return false;
}
05
Kiválasztás
Megkeresi az első 100-as értékű elem indexét (feltételezzük, hogy biztosan van ilyen).
static int Kivalasztas(int[] T)
{
    for (int i = 0; i < T.Length; i++)
        if (T[i] == 100)
            return i;
    return -1;
}
