WinForms – Label Grid (bool[,])

NumericUpDown · Label tömb · véletlenszerű zöld/piros rács — hiányzik a portfólióból
01
Form1.cs – teljes kód
Két NumericUpDown (sor/oszlop), Panel-re dinamikusan elhelyezett Label tömb, véletlenszerű logikai értékekkel.
public partial class Form1 : Form
{
    Label[,] cimkek;
    bool[,]   logikai;
    int sor, oszlop;

    public Form1()
    {
        InitializeComponent();
    }

    private void numericUpDown1_ValueChanged(object sender, EventArgs e)
    {
        sor    = (int)numericUpDown1.Value;
        oszlop = (int)numericUpDown2.Value;

        panel1.Controls.Clear();
        cimkek  = new Label[sor, oszlop];
        logikai = new bool[sor, oszlop];

        Random r = new Random();

        for (int i = 0; i < sor; i++)
        {
            for (int j = 0; j < oszlop; j++)
            {
                logikai[i, j] = r.Next(2) == 1;

                cimkek[i, j] = new Label();
                cimkek[i, j].Width       = 40;
                cimkek[i, j].Height      = 40;
                cimkek[i, j].Left        = j * 42;
                cimkek[i, j].Top         = i * 42;
                cimkek[i, j].BorderStyle = BorderStyle.FixedSingle;
                cimkek[i, j].BackColor   = logikai[i, j]
                    ? Color.Lime
                    : Color.Red;

                panel1.Controls.Add(cimkek[i, j]);
            }
        }
    }
}
⚠️ WinForms projekt. A numericUpDown1 és numericUpDown2 ValueChanged eseményét ugyanahhoz a handlerhez kell kötni. A panel1 AutoScroll = true legyen, ha nagy rácsot szeretnél.
