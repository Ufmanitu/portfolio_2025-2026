Rendezési algoritmusok

ECS · BUBI · MinMax · Beszúró · Shell — hiányzik a portfólióból
01
Egyszerű cserés rendezés (ECS)
Minden i-re megkeresi a tömb többi elemét, és felcseréli, ha szükséges. O(n²)
static void ECS(int[] T)
{
    int seged;
    for (int i = 0; i < T.Length; i++)
    {
        for (int j = i + 1; j < T.Length; j++)
        {
            if (T[i] > T[j])
            {
                seged = T[i];
                T[i] = T[j];
                T[j] = seged;
            }
        }
    }
}
02
Buborékos rendezés (BUBI)
Szomszédos elemeket hasonlít össze és cseréli fel. Minden menetben a legnagyobb „buborékként" felúszik a végére. O(n²)
static void BUBI(int[] T)
{
    int seged;
    for (int i = T.Length - 1; i > 0; i--)
    {
        for (int j = 0; j < i; j++)
        {
            if (T[j] > T[j + 1])
            {
                seged = T[j];
                T[j] = T[j + 1];
                T[j + 1] = seged;
            }
        }
    }
}
03
MinMax (kiválasztásos) rendezés
Minden menetben megkeresi a legkisebb elemet és a helyére teszi. O(n²)
static void MinMax(int[] tomb)
{
    int index, seged;
    for (int i = 0; i < tomb.Length; i++)
    {
        index = i;
        for (int j = i + 1; j < tomb.Length; j++)
        {
            if (tomb[index] > tomb[j])
                index = j;
        }
        seged = tomb[i];
        tomb[i] = tomb[index];
        tomb[index] = seged;
    }
}
04
Beszúró rendezés
Minden elemet a már rendezett részbe illeszti be a megfelelő helyre. O(n²), de kis adatoknál gyors. Stabil.
static void BeszuroR(int[] tomb)
{
    int kulcs, j;
    for (int i = 1; i < tomb.Length; ++i)
    {
        kulcs = tomb[i];
        j = i - 1;

        while (j >= 0 && tomb[j] > kulcs)
        {
            tomb[j + 1] = tomb[j];
            j--;
        }
        tomb[j + 1] = kulcs;
    }
}
05
Shell rendezés
A beszúró rendezés általánosítása: csökkenő „gap" értékekkel dolgozik. Közelítőleg O(n log²n).
static void Shell(int[] tomb)
{
    int gap, j, temp;

    for (gap = tomb.Length / 2; gap > 0; gap /= 2)
    {
        for (int i = gap; i < tomb.Length; i++)
        {
            temp = tomb[i];
            j = i;

            while (j >= gap && tomb[j - gap] > temp)
            {
                tomb[j] = tomb[j - gap];
                j -= gap;
            }
            tomb[j] = temp;
        }
    }
}
