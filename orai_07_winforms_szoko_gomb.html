Form1.cs – szökő gomb kódja
Amikor az egér a gomb fölé kerül (MouseMove), a gomb véletlenszerű pozícióba ugrik az ablakon belül.
public partial class Form1 : Form
{
    Random r = new Random();

    public Form1()
    {
        InitializeComponent();
    }

    private void button1_MouseMove(object sender, MouseEventArgs e)
    {
        button1.Left = r.Next(0, this.Width  - button1.Width);
        button1.Top  = r.Next(0, this.Height - button1.Height);
    }
}
⚠️ WinForms projekt. Elég egyetlen Button a Designerben. A MouseMove eseményt kell beállítani (nem Click!). Az ablak FormBorderStyle = FixedSingle és MaximizeBox = false javasolt.
