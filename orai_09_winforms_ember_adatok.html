WinForms – Ember adatbázis (kor + fizetés)

ComboBox · List<Ember> · fájlból olvasás — hiányzik a portfólióból
ℹ️ A portfólió #14-es feladata (Telefonregiszter) szintén ComboBox + fájlolvasás, de Nev, Varos, Szam mezőkkel. Ez az órai változat Nev, Kor, Fizetes adatokat használ, az adatok.txt fájlból olvasva.
01
Form1.cs – teljes kód
Ember osztály Nev/Kor/Fizetes tulajdonságokkal, fájlból töltés, ComboBox szelekció esemény.
public partial class Form1 : Form
{
    class Ember
    {
        public string Nev     { get; set; }
        public int    Kor     { get; set; }
        public int    Fizetes { get; set; }
    }

    List<Ember> lista = new List<Ember>();

    public Form1()
    {
        InitializeComponent();
        Beolvas();
        Feltolt();
    }

    void Beolvas()
    {
        foreach (var sor in File.ReadAllLines("adatok.txt"))
        {
            var s = sor.Split(';');
            lista.Add(new Ember()
            {
                Nev     = s[0],
                Kor     = int.Parse(s[1]),
                Fizetes = int.Parse(s[2])
            });
        }
    }

    void Feltolt()
    {
        foreach (var e in lista)
            comboBox1.Items.Add(e.Nev);
    }

    private void comboBox1_SelectedIndexChanged(object sender, EventArgs e)
    {
        var ember = lista.Find(x => x.Nev == comboBox1.Text);
        label1.Text = ember.Kor.ToString();
        label2.Text = ember.Fizetes.ToString();
    }
}
⚠️ WinForms projekt. Szükséges: comboBox1, label1 (kor), label2 (fizetés). Az adatok.txt formátuma: Neve;30;450000 soronként. Másolj a projekt bin/Debug mappájába.
